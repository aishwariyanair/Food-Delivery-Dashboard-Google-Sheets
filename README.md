#Swiggy Sales Dashboard
 Overview

This project presents an interactive sales dashboard built using Google Sheets to analyze food delivery data. It highlights key performance metrics, trends, and business insights.
# Key Features
-KPI metrics (Total Revenue, Average Rating, Average Price, Total Orders)
-Monthly trend analysis of orders
-Top 5 categories by order volume
-Top 5 locations contributing to revenue

# Tools Used
Google Sheets
Pivot Tables
Data Visualization

# Insights
Recommended category has highest number of orders
Majority of revenue comes from top locations
Monthly trends show fluctuations in order volume
